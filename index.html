Installation Guide & FAQ

How to install the extension ▼
  1. Download the extension ZIP file by clicking the download button
  2. Extract the ZIP file to a folder on your computer
  3. Open Chrome and go to chrome://extensions/
  4. Enable "Developer mode" in the top right corner
  5. Click "Load unpacked" and select the extracted folder
  6. The extension will now be active and protecting your browsing
How it works ▼

Our extension uses advanced content filtering algorithms that run entirely on your device. It scans webpage content in real-time and automatically blurs or blocks harmful material. The filtering happens instantly as pages load, ensuring children are protected from inappropriate content without any delays or interruptions to their browsing experience.


Key Technologies:

  • Machine learning-based text analysis
  • Image recognition for visual content
  • Real-time DOM manipulation
  • Local processing for privacy
How to report content ▼

If you encounter harmful content that wasn't filtered, or if you need to report serious incidents:


  1. Use our reporting form above - Fill out the incident details
  2. Contact local authorities - Use the Kerala emergency contacts provided
  3. Take screenshots - Document the harmful content (safely)
  4. Note the URL - Record where the content was found
  5. Report to platform - Also report directly to the website/platform

Remember: If there's immediate danger, always call emergency services first (100 in India).
